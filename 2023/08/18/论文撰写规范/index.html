论文撰写规范

发表于 2023-08-18 本文字数： 233 阅读时长 ≈ 1 分钟

语言规范

  1. 文中尽量少用主动语态，多用被动语态。

    e.g. We used the model to predict epilepsy. $\to$ The model was used to predict epilepsy.

数学公式规范

参考文章链接：https://www.renrendoc.com/paper/148467632.html

正体

  1. 有固定定义的函数

    如：${\rm sin(), cos(), tan(), ln(), log(), sgn()}$

  2. 有特定意义的缩写

    如：${\rm max, min}$

  3. 运算符号

    如：$\sum, \Pi$

  4. 特殊集合符号（加粗正体）

    如：$\rm \textbf{N}, \textbf{Z}, \textbf{R}$

  5. 计量单位中的代号

    如：$\rm cm, m, kg, \Omega$

  6. 表示命题条件的字母
  7. 表示序号的拉丁字母
  8. 其值不变的数学常数符号

    如：$\pi, i(虚数单位)$

斜体

  1. 表示变量和函数的字母

    如：$x,y,f(x)$

  2. 表示参数的字母
  3. 表示几何图形中点、线、面、体的字母
  4. 表示坐标系符号
  5. 表示向量、矩阵符号（斜体加粗）

    如：矩阵A
